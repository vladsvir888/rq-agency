## Available Scripts

```bash

npm run component [unit-name] [component-name]               # Utility for creating component. Unitname: @ (atoms), ^ (molecules) or % (organisms). Componentname is a component that begins with app-. This utility creates a folder with a component and imports .scss file into main.scss
```
